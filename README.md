# Food Calorie Counter

A simple Python program that allows users to log food items they consumed and calculates total calorie intake based on predefined calorie values.

## Description

This project helps users keep track of their calorie consumption by entering food items and the number of servings. It includes common foods plus some popular Indian dishes with their calorie values. The program calculates and sums the calories for all logged items.

## Features

- Supports multiple food items with calorie calculation.
- Includes Indian dishes like dal, roti, paneer, samosa, idli, vada, biryani, and chai.
- Handles invalid entries and improper inputs gracefully.
- Command-line interface for ease of use.

## Installation

Make sure you have Python 3 installed on your system.
No external libraries are required.

## Usage

Run the script from your terminal or command prompt:


Enter food items one by one and specify servings.
Type `done` to finish and see the total calories consumed.
